# Leon 🎨

🌐 **Live Demo:** https://leon-template-987.pages.dev

🖼️**PSD UI File Reference:** https://www.graphberry.com/item/leon-psd-agency-template

A minimal digital agency landing page built using **HTML5 and CSS3**.

The project presents a fictional creative agency site called "Leon," featuring a hero landing section, service highlights, a portfolio showcase, an about section, and a contact section — all in a single responsive page.

## Overview

Leon is a front-end template project created to practice translating a design into a fully responsive, styled webpage using real CSS, moving beyond the raw-HTML-only projects in this series.

The goal of this project was to focus on:
- CSS3 fundamentals (box model, flexbox, CSS grid)
- Responsive design with media queries
- CSS custom properties (variables)
- Pseudo-elements and hover interactions
- Google Fonts and icon font integration
- Structuring a real single-page marketing site

## Features

### 🧭 Header & Navigation
- Fixed logo and a dropdown-style mobile menu using a hamburger icon (CSS-only, no JavaScript)
- Smooth-scroll anchor links to each section

### 🌄 Landing / Hero
- Full-height hero section with background image and centered intro text

### ✨ Features Section
- Three-column responsive grid highlighting the agency's process, using Font Awesome icons

### 🛠️ Services
- Two-column service list (Graphic Design, UI & UX, Web Design, Web Development) alongside a styled image with layered CSS pseudo-elements

### 🖼️ Portfolio
- Responsive card grid showcasing sample projects with images and descriptions

### 👤 About
- Two-column layout combining an image with decorative CSS borders and descriptive text

### ✉️ Contact
- Centered contact section with a mailto link and social media icon links

### 🦶 Footer
- Simple centered footer with copyright text

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, custom properties, media queries, pseudo-elements)
- Font Awesome (icon font)
- Google Fonts (Work Sans)

No JavaScript was used; the mobile menu interaction is handled entirely with CSS `:hover`.

## CSS Techniques Practiced

This project uses a range of modern CSS3 techniques:
- CSS custom properties (`:root` variables for colors, spacing, transitions)
- Flexbox for header and about layout
- CSS Grid for features, services, and portfolio layouts
- Responsive breakpoints via `@media` queries
- `::before`/`::after` pseudo-elements for decorative shapes
- CSS transitions for hover states and the mobile menu
